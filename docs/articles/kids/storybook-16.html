📡 ESipa: The Superhighway

The busiest bridge: connecting the Control Centre to the device. It carries commands, receives results, checks status, triggers downloads, delivers reports. Five different gadgets in one interface! The heartbeat of IoT eSIM communication.

Page 5 of 14

🏭 ES9+': Factory Hotline

A direct line from the Control Centre to the Profile Factory. When a sleepy device can't reach the factory, the Control Centre does it instead. Authenticate, download, cancel: all handled by remote proxy!

Page 7 of 14

📮 ES11': Board Checker

The Control Centre uses this to check the Message Board on behalf of sleepy devices. The device never talks to the board at all: the Control Centre checks, finds any waiting notes, and forwards them. Zero extra battery drain!

🔐 ESep: The Secret Tunnel

The most important gadget! It's a logical tunnel inside the main highway. Every command gets two signatures: one from the Control Centre (who sent it) and one from the chip (who executed it). Double-proof that everything is legitimate!

🆕 Eight New Gadgets

SGP.32 added eight new chip-level commands! Load signed packages, set up the first trusted manager, read the contact list, rollback bad changes, instant activation, and the emergency parachute: a fallback profile that activates if everything else fails.
